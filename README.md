## Getting Started

To get a local copy up and running follow these simple example steps.

```
git clone https://github.com/Hector096/crazy-cryptos.git
```

and then run

```
npm i
npm start
```

## Installation

to build a production version to `bulid` folder run

```
npm run build
```
